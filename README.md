# CommunityStudy
# 问答社区开发学习

对该问答社区的开发和学习使本人对于Web项目的后端开发中的开发流程、数据库、框架等技术的使用有了更深入的了解，收获颇多。

## 功能列表

注册、登录、在线问答、关注、点赞、通知、私信，搜索等功能。

## 开发流程

1.开发社区首页

2.开发注册，邮件发送，激活

3.开发登录，账号设置，验证码

4.过滤敏感词

5.开发发布帖子，帖子详情

6.开发评论和回复功能

7.开发点赞、关注

8.开发社区搜索功能
